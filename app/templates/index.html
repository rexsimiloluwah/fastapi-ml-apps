Machine learning apps using FastAPI

A collection of some basic Machine learning apps built using FastAPI

01. Heart Disease Prediction

A Machine learning binary classifier for predicting the possibility of a heart disease from several features.

02. Sentiment Analysis (NLP)

Sentiment Analysis for Movie Reviews to classify the movie review into a positive or negative sentiment.

03. Computer Vision API

Deploying a Computer Vision API using FastAPI for multi-class classification.

04. Object Detection API

Object Detection API based on SSD and OpenCV DNN module.
